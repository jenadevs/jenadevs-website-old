Values

Based on the Manifesto for Software Craftsmanship and Agile Software Development as well as our own empirical evidence.

Individuals & Interactions

Every human being is unique and brings his own skill set to the table. We value self-organisation, collaboration and open exchange.

Working Software

Is more valueable than producing other artifacts.

Code & Tools

Well-written and -tested code is a sign of true craftsmanship. A good craftsman always picks the right tool for the job.

Continuous Development

Does not only apply to software, but also to processes, tools, skills and the personal level.
