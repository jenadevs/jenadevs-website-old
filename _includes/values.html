Values

Based on the Manifesto for Software Craftsmanship and Agile Software Development as well as our own empirical evidence.

Individuals & Interactions

Every human being is unique and thus may develop his own set of craftsman skills. We value self-organisation, a sense for collaboration and open exchange.

Working Software

Functional Software is more valueable than producing other artifacts, like documents. The domain drives the value and enforces to quickly respond to changing requirements.

Code & Tools

Clean code does not exist without tests. A sign of true craftsmanship is to always pick the fitting tool set for the project. However: Focus on people, not tools.

Continuous Development

Not only software, but also processes, tools, skills as well as individuals need and should always follow the path of constant improvement. Standstills are always steps backwards.
